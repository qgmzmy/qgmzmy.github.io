通过线刷简单获取Root

  1. 刷机的第一步要先解锁Bootloader，目前也并不是所有手机都支持解锁，具体需看各家厂商对刷机的态度，各家的手机解锁方法也略有不同。另外，解锁Bootloader会清空手机数据。

    2. 品牌 是否可解锁 解锁方法
    小米 是 miflash_unlock
    红米 是 miflash_unlock
    维沃 否 无
    iQOO 否 无
    欧珀 部分可解锁 深度测试
    真我 部分可解锁 深度测试
    一加 是 Android 调试桥
    谷歌 部分可解锁 Android 调试桥
    魅族 否 无
    三星 是 未知
    索尼 部分可解锁 解锁码
    摩托罗拉 是 解锁码
    华为 否 解锁码
    HTC 是 未知
    荣耀 否 未知
    努比亚 是 未知

  2. 当你解锁了Bootloader后就可以在手机上下载并安装Magisk或Magisk Delta了

  3. 接下来为了修补boot需要获取boot镜像，一般我们会从当前系统的卡刷包或者线刷包中提取

    1. 首先需要下载当前系统的卡刷包或者线刷包（一定要是当前的系统版本！！！）

    2. 解压

    3. 若刷机包内有boot.img就直接拿出来备用，若没有boot.img但有payload.bin可使用payload-dumper-go提取boot.img（使用payload-dumper-go提取payload.bin中的boot.img）

  4. 将提取出的boot.img传输到手机的内部存储中


  5. 在手机中打开Magisk或Magisk Delta

  6. 点击Magisk旁的安装

  7. 选择“选择并修补一个文件”

  8. 选择boot.img并点击开始

  9. 将修补后的boot从Download移动到电脑


  10. 打开设置

  11. 打开关于手机

  12. 连续点击版本号直至打开开发者模式

  13. 在开发者选项中打开“USB 调试”

  14. 在电脑上下载platform-tools并解压

  15. 在解压后得到的文件夹中打开cmd


  16. 在cmd中输入：

    17. adb reboot bootloader

  17. 进入Fastboot后输入：

    18. fastboot flash boot <Boot Image>

    此处“<Boot Image>”表示修补后的boot镜像。


  18. 重启手机，Root成功
